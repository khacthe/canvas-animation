# Craw canvas 2D, 3D, Animation with P5 library

## Add library
- Quản lý thư viên với npm
```
npm install -g p5-manager // install p5 manager

// Initialize a new collection

p5 new my_project

// Generate p5 project

p5 g my_project

// Generate p5 project and bundle

p5 g -b my_project

// Run server

p5 server

OR

p5 s

```
